## 模组简介

开局1个抽奖机，物资装备全靠抽。听说你是欧皇？来试试你能活几天……  
<small>_PS：新游戏 -> 船难(DLC) -> 世界(设定) -> 荒岛求生(预设)_</small>

### 设定

- 抽奖机：调整抽奖机的奖励、概率等机制。  

- 投入金币 = 抽奖 + 伤脑(-5理智)  

- 投入物品 = 金块->1板条箱、宝石->2金块、蔬果->2金币、其他->1理智  

- 金块奖项：新增[2渡渡鸟宝宝]；调整[1高礼帽、1缝纫包、3布]->抽奖概率降低为1；  

- 萝卜奖项：新增[3藤 + 1藤蔓根]、[3海带 + 1海带根]、[3石灰石 + 1珊瑚幼虫]、[4蜜蜂 + 1蜂巢]；  

- 骷髅奖项：调整[2猎犬]->[1鳄犬 + 50%冰鳄犬/50%毒鳄犬]；调整[3蜘蛛]->[2蜘蛛 + 50%蜘蛛/50%战士 + 1%女皇]；

- 配方：  
  - 精炼栏配方：新增[1金块 -> 2金币]；调整[金块]->无需科技。  
  - 工具栏配方：调整[燧石工具]->无需科技。

- 地皮：调整鸟类的刷新机制。  
  - 沙地(大嘴鸟、鹦鹉、海盗鹦鹉)  
  - 木地板(鹦鹉、海盗鹦鹉)  
  - 草地(海盗鹦鹉)

- 猎犬袭击：强制设定为最高级别(增加荒岛生存乐趣)。

- 种树：植树造林可以改善沙地(树木生长会将沙地变为草地)。

- 蔬果：(有种子的)蔬菜水果腐烂后可以获得1个种子。

- 渡渡鸟：渡渡鸟在物品栏喂养时，可以获得种子(喂菜)或鸟粪(喂肉)。

- 海草：月圆之夜被采摘，再生时则有50%几率会变异为芦苇。

- 便便农场：腐烂食物、便便、鸟粪可以播种，收获后的农场可以铲子挖起来。  
  - 便便：作物2天成熟，肥力初始10点，上限20点，肥力归零时消耗1个。  
  - 鸟粪：作物1天成熟，肥力初始12点，上限30点，肥力归零时消耗1个。  
  - 可以用“腐烂作物”施肥，种植后需要用铲子翻土(有几率获得：50%石头、20%燧石、16%硝石、13%石灰石、1%鼹鼠)。

- 拉便便：人物吃撑(>99%)后不能再吃东西；傍晚时会拉1便便；拉完便便后可以吃东西；拉便便时可能会有一些未消化的种子。

- 贝壳：沙滩上会生成贝壳、板条箱、漂流瓶。

- 板条箱：砸开会获得1木板、1贝壳，随机1个物资：(18%)竹根、(18%)藤蔓根、(18%)草丛、(18%)树苗、(18%)鱼罐头、(9%)漂流瓶、(1%)金币。

- 鱼罐头：开启后，随机1个食物：(36%)熟鱼肉、(36%)熟大肉、(9%)熟石斑鱼、(9%)熟小丑鱼、(9%)熟霓虹鱼、(1%)曼德拉汤。

- 漂流瓶：阅读后获得空瓶子、藏宝图(70%纸、30%蓝图)。  

- 砍刀：可以将木头、椰树叶、芦苇、大肉、鱼块分解(与椰子类似)。  

欢迎反馈问题、提供有趣的设定。
